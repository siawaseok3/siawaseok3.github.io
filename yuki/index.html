YukiYouTubeの作り方

初めに

  1. RenderとGitHubまたはGitLabでアカウントを作成。
  2. RenderとGitHubまたはGitLabのアカウントを紐付けておくと便利です。

リポジトリの作成

GitHubの場合

  1. https://github.com/siawaseok3/slim-2-by-siawaseokかhttps://github.com/mochidukiyukimi/yuki-youtube-slim-2.git(公式)を開く。
  2. リポジトリをフォークする。
    （偽装ページを変更したい場合は、自分がフォークしたリポジトリのblog/index.htmlファイルを編集してください）

GitLabの場合

  1. https://gitlab.com/projects/new#import_projectを開く → URLごとのリポジトリを選択。
  2. https://github.com/siawaseok3/yuki-by-siawaseok.git か
    https://github.com/mochidukiyukimi/yuki-youtube-slim-2.git をGitリポジトリのURLって所に貼り付ける
  3. プロジェクト名に半角英数で好きな文字をいれる
  4. プロジェクトを作成。

デプロイ

  1. Render Web Serviceを開く。
  2. RenderとGitHubまたはGitLabのアカウントを紐付けている場合は、先ほどフォークしたリポジトリを選択。
  3. その他の場合は、Public Git Repositoryを選択してリポジトリのURLを貼り付け、

    Connectを押す。
  4. Name欄の文字を消して、好きな名前に変更（半角英数とハイフンのみ使用可）。
  5. Start Commandにuvicorn --port $PORT --host 0.0.0.0 main:appを入力。
  6. Instance Typeが有料プランになっているので、Freeを選択。
  7. Deploy Web Serviceを押す。
  8. デプロイが完了するまで待機。
